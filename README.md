## Overview

This project is an **HR Analytics Dashboard** designed to provide insights into workforce data through interactive visualizations and key performance indicators (KPIs). The dashboard helps HR teams and management monitor employee metrics, analyze workforce trends, and support data-driven decision-making.

## Features

* Employee headcount analysis
* Department-wise employee distribution
* Gender diversity insights
* Employee attrition tracking
* Age and experience analysis
* Salary distribution overview
* Interactive filters and visualizations
* Performance and workforce trend monitoring
